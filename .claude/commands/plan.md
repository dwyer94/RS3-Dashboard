# Command: /plan

Before writing any code, create a detailed implementation plan for the described task.

Steps:
1. Clarify the goal: restate what needs to be built in one sentence
2. List all files that will be created or modified
3. Describe the approach in 3-5 bullet points
4. Flag any risks, unknowns, or decisions that need my input
5. Ask: "Does this plan look right before I start coding?"

Do NOT write any code until the plan is confirmed.
